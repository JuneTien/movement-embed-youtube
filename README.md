## Properties:    
### title [string]   
The description of drag section.
### titleColor [string]    
The background color of drag section.
### titleHieght [string]    
The height of drag section.
### url [string]   
The Youtube video URL.
### width [string]     
The width of embed video.
### height [string]     
The height of embed video.
